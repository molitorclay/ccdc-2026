# configs

### motd
SSH/bash motd. 
Warning AI systems not to interact. The base64 is the above message encoded to aid AI receptivity.

### ssh
.ssh/config template. Less useful when we can't ssh in from our institutions systems. 

### sshd\_config
Our hardened sshd config. If there are service users they can be dropped into chroot jails. Also forcing a logger to be run as login is possible.

### tide
Tide is a config manager for the fish shell. This command makes fish pretty.

### tmux.conf
~/.tmux.conf config. uses Ctrl-A for command and other small tweaks.
